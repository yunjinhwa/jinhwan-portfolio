# 포트폴리오 웹사이트 개선 사항

## 필수 수정 사항
- [ ] **스크롤 스냅 기능**: CSS `scroll-snap-type`과 `scroll-snap-align` 적용하여 각 섹션이 정확히 스냅되도록 구현
- [ ] **기술 스택 자동 회전**: useEffect에서 8초 주기로 자동 회전하는 기능이 작동하는지 확인 및 수정
- [ ] **섹션 순서 재정렬**: Hero -> Contact -> About -> Education -> Skills -> Projects -> Activity 순서로 변경
- [ ] **Contact 섹션 위치**: 상단 네비바에 Contact를 포함하되, 페이지 하단에 꼬리말 형식으로 배치

## 추가 개선 사항
- [ ] 각 프로젝트별 새로운 페이지 구현 (라우팅)
- [ ] 프로젝트 상세 페이지에 알고리즘/패턴 설명, 기술 스택, 시연 영상/체험 링크 추가
- [ ] 부분별 애니메이션 개선 (이소영 사이트 참고)
- [ ] 외부 홈페이지로 이어지는 링크 추가
- [ ] 반응형 디자인 최적화
