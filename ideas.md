# 윤진환 포트폴리오 웹사이트 디자인 기획

## 선택된 디자인 철학: 파스텔 미니멀리즘 (Pastel Minimalism)

### 디자인 무브먼트
**파스텔 톤 미니멀리즘**: 부드럽고 따뜻한 파스텔 색상을 기반으로 한 현대적 미니멀 디자인. 심플하면서도 세련된 느낌으로 개발자의 전문성을 표현합니다.

### 핵심 원칙
1. **심플함 속의 우아함**: 불필요한 요소를 제거하되, 중요한 콘텐츠가 자연스럽게 돋보이도록 설계
2. **부드러운 인터랙션**: 스크롤 스냅, 카드 호버 효과 등으로 사용자 경험을 부드럽게 연결
3. **명확한 시각 계층**: 타이포그래피와 공간을 통해 정보의 우선순위를 명확히 표현
4. **따뜻한 감정**: 파스텔 톤으로 접근성 높고 친근한 느낌 전달

### 색상 철학
**Coolors.co 파스텔 드림랜드 어드벤처** 팔레트 기반:
- **배경**: 밝은 크림색/화이트 (밝고 편안한 느낌)
- **포인트 컬러**: 파스텔 핑크, 파스텔 블루, 파스텔 라벤더, 파스텔 민트
- **텍스트**: 진한 슬레이트 그레이 (명확한 가독성)
- **강조**: 파스텔 핑크/블루 (CTA 버튼, 활성 상태)

### 레이아웃 패러다임
- **세로 스크롤 중심**: 자연스러운 스토리텔링 흐름
- **스크롤 스냅**: 각 섹션이 명확하게 구분되는 느낌
- **카드 기반 구성**: 프로젝트, 경력, 활동을 카드로 표현
- **상단 네비바**: 책갈피처럼 빠른 이동 가능

### 시그니처 요소
1. **파스텔 그래디언트 배경**: 섹션별로 미묘한 그래디언트 (예: 핑크→라벤더)
2. **둥근 카드 컴포넌트**: border-radius로 부드러운 느낌
3. **아이콘 회전 애니메이션**: Skills 섹션의 기술 스택 아이콘 자동 회전

### 인터랙션 철학
- **호버 효과**: 카드가 살짝 떠오르는 느낌 (shadow + scale)
- **스크롤 트리거**: 요소가 스크롤 시 부드럽게 나타나는 효과
- **클릭 피드백**: 버튼 클릭 시 명확한 시각적 반응

### 애니메이션 가이드
- **진입 애니메이션**: fade-in + slide-up (200-300ms)
- **호버 상태**: scale(1.02) + shadow 증가 (150ms)
- **스크롤 스냅**: smooth scroll behavior
- **아이콘 회전**: 자동 회전 (8초 주기), 클릭으로 수동 제어

### 타이포그래피 시스템
- **디스플레이 폰트**: 고딕 계열 (예: Pretendard Bold) - 제목, Hero 섹션
- **본문 폰트**: 고딕 계열 (예: Pretendard Regular) - 본문, 설명
- **계층 구조**:
  - H1: 48px, Bold (Hero 제목)
  - H2: 32px, Bold (섹션 제목)
  - H3: 24px, SemiBold (카드 제목)
  - Body: 16px, Regular (본문)
  - Caption: 14px, Regular (부가 정보)

## 섹션별 디자인 계획

### Hero 섹션
- 큰 타이포그래피로 "윤진환"과 한 줄 소개 표현
- 파스텔 그래디언트 배경
- 부드러운 스크롤 다운 유도 화살표

### Contact (상단 네비바)
- 고정 위치 네비바
- 각 섹션으로의 빠른 이동 링크
- 이메일, 깃허브 등 연락처 링크

### About 섹션
- 자기소개 텍스트
- 개발자로서의 가치관 표현
- 프로필 이미지 (선택)

### Education 섹션
- 학력 정보 카드 형식
- 연도별 정렬

### Skills 섹션
- 기술 스택 아이콘 (자동 회전)
- 클릭으로 수동 제어
- 카테고리별 분류 (Frontend, Backend, Tools 등)

### Projects 섹션
- 각 프로젝트별 카드
- 프로젝트 개요, 사용 기술, 링크
- 클릭 시 상세 페이지로 이동

### Activity 섹션
- 활동 내역 카드
- 수상, 참여 프로젝트 등

### Contact (하단)
- 꼬리말 형식의 연락처
- 이메일, 깃허브, 링크드인 등
